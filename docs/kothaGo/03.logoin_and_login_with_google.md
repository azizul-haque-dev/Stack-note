# Part 3 — Login & Sign in with Google
## POST /api/v1/auth/login · POST /api/v1/auth/google

---

## Flows

### Credentials Login
```
Client → POST /auth/login { email, password }
    ↓
Rate limiter (10 req/min per IP)
    ↓
Zod validation
    ↓
Find user by email — also fetch passwordHash (explicitly selected)
    ↓
bcrypt.compare (runs EVEN if user not found — prevents timing attack)
    ↓
Check user exists + password valid + isActive + provider === local
    ↓
Generate access + refresh tokens
    ↓
Web: HttpOnly cookie + { accessToken, user }
Mobile: { accessToken, refreshToken, user }
```

### Google OAuth
```
Frontend → Google OAuth → receives id_token (JWT signed by Google)
    ↓
Client → POST /auth/google { idToken }
    ↓
Rate limiter (20 req/min per IP)
    ↓
Backend verifies id_token via google-auth-library
    ↓
Extract: googleId, email, name, picture, email_verified
    ↓
Find user by googleId
    ↓ (not found)
Check if email already exists with local provider → reject with clear message
    ↓ (ok)
Create new Google user
    ↓
Generate access + refresh tokens
    ↓
Web: HttpOnly cookie + { accessToken, user }
Mobile: { accessToken, refreshToken, user }
```

## 1. Auth Service — login() + googleAuth()

```typescript
// backend/src/auth/auth.service.ts
import * as bcrypt from 'bcrypt';
import { OAuth2Client } from 'google-auth-library';
import { AuthProvider } from '@prisma/client';
import { v4 as uuidv4 } from 'uuid';
import { usersService } from '../users/users.service';
import { tokensService } from '../tokens/tokens.service';

const googleClient = new OAuth2Client(process.env.GOOGLE_CLIENT_ID);

type Meta = { userAgent?: string; ipAddress?: string };

function sanitizeUser(user: any) {
  const { passwordHash, ...safe } = user;
  return safe;
}

export const authService = {
  // register() — see Part 2

  // ─── Login (credentials) ─────────────────────────────────────────────────────
  async login(data: { email: string; password: string }, meta: Meta = {}) {
    // fetchs user WITH passwordHash (explicitly selected — omitted by default)
    const user = await usersService.findByEmailWithPassword(data.email);

    // ── Timing attack prevention ──────────────────────────────────────────────
    // If user doesn't exist, we still run bcrypt.compare against a dummy hash.
    // Without this, a non-existent email returns instantly (no bcrypt work),
    // while a valid email takes ~300ms. Attackers can enumerate valid emails
    // by measuring response time — known as a timing attack.
    const DUMMY_HASH = '$2b$12$dummyhashforsecuritypurposesXXXXXXXXXXXXXXXX';
    const hashToCompare = user?.passwordHash ?? DUMMY_HASH;
    const isPasswordValid = await bcrypt.compare(data.password, hashToCompare);

    // Single generic error — never reveal whether email or password was wrong
    if (!user || !isPasswordValid || !user.isActive) {
      const err: any = new Error('Invalid credentials');
      err.status = 401;
      throw err;
    }

    // Prevent local login for Google accounts
    if (user.provider !== AuthProvider.local) {
      const err: any = new Error(
        `This account was created with Google. Please sign in with Google.`,
      );
      err.status = 400;
      throw err;
    }

    const family = uuidv4();
    const accessToken = tokensService.generateAccessToken({ sub: user.id, email: user.email });
    const refreshToken = await tokensService.generateRefreshToken(
      user.id, user.email, family, meta,
    );

    return { accessToken, refreshToken, user: sanitizeUser(user) };
  },

  // ─── Google OAuth ─────────────────────────────────────────────────────────────
  async googleAuth(idToken: string, meta: Meta = {}) {
    // ── Step 1: Verify the ID token with Google ───────────────────────────────
    // This hits Google's API internally to validate the token's signature and claims.
    // It also checks: aud (matches our client ID), exp (not expired), iss (Google).
    let ticket;
    try {
      ticket = await googleClient.verifyIdToken({
        idToken,
        audience: process.env.GOOGLE_CLIENT_ID,
      });
    } catch {
      const err: any = new Error('Invalid Google token');
      err.status = 401;
      throw err;
    }

    // ── Step 2: Extract claims ────────────────────────────────────────────────
    const payload = ticket.getPayload();
    if (!payload?.email) {
      const err: any = new Error('Could not retrieve email from Google token');
      err.status = 401;
      throw err;
    }

    const {
      sub: googleId,        // Stable Google user identifier
      email,
      given_name,
      family_name,
      picture,
      email_verified,
    } = payload;

    // ── Step 3: Find or create user ───────────────────────────────────────────
    let user = await usersService.findByGoogleId(googleId);

    if (!user) {
      // Check if this email was already registered with credentials
      const existingLocal = await usersService.findByEmail(email);
      if (existingLocal && existingLocal.provider === AuthProvider.local) {
        // Don't auto-link silently — that's a security risk.
        // Force the user to log in with their password and optionally link accounts later.
        const err: any = new Error(
          'An account with this email already exists. Please log in with your password.',
        );
        err.status = 409;
        throw err;
      }

      // New Google user — create their account
      user = await usersService.create({
        email: email.toLowerCase().trim(),
        googleId,
        firstName: given_name,
        lastName: family_name,
        avatarUrl: picture,
        provider: AuthProvider.google,
        emailVerified: email_verified ?? false,
        passwordHash: null, // Google users have no password
      });
    }

    if (!user.isActive) {
      const err: any = new Error('Account is deactivated');
      err.status = 401;
      throw err;
    }

    // ── Step 4: Issue tokens ──────────────────────────────────────────────────
    const family = uuidv4();
    const accessToken = tokensService.generateAccessToken({ sub: user.id, email: user.email });
    const refreshToken = await tokensService.generateRefreshToken(
      user.id, user.email, family, meta,
    );

    return { accessToken, refreshToken, user: sanitizeUser(user) };
  },
};
```

## 2. Auth Router — /login, /google, /logout, /me

```typescript
// backend/src/auth/auth.router.ts
import { Router, Request, Response, NextFunction } from 'express';
import { authService } from './auth.service';
import { tokensService } from '../tokens/tokens.service';
import { usersService } from '../users/users.service';
import { validate, loginSchema, googleAuthSchema } from './auth.validators';
import {
  authenticate,
  authenticateRefresh,
  AuthRequest,
} from '../middleware/authenticate';
import { loginLimiter, googleLimiter } from '../middleware/rate-limit';

const router = Router();

const IS_PRODUCTION = process.env.NODE_ENV === 'production';

const REFRESH_COOKIE_OPTIONS = {
  httpOnly: true,
  secure: IS_PRODUCTION,
  sameSite: 'strict' as const,
  maxAge: 7 * 24 * 60 * 60 * 1000,
  path: '/api/v1/auth',
};

function isMobile(req: Request): boolean {
  return /ReactNative|Expo|okhttp/.test(req.headers['user-agent'] ?? '');
}

function getMeta(req: Request) {
  return {
    userAgent: req.headers['user-agent'],
    ipAddress: (req.headers['x-forwarded-for'] as string) ?? req.ip,
  };
}

// ─── POST /auth/login ────────────────────────────────────────────────────────
router.post(
  '/login',
  loginLimiter,
  validate(loginSchema),
  async (req: Request, res: Response, next: NextFunction) => {
    try {
      const { accessToken, refreshToken, user } = await authService.login(
        req.body,
        getMeta(req),
      );

      if (!isMobile(req)) {
        res.cookie('refresh_token', refreshToken, REFRESH_COOKIE_OPTIONS);
        return res.json({ accessToken, user });
      }

      return res.json({ accessToken, refreshToken, user });
    } catch (err) {
      next(err);
    }
  },
);

// ─── POST /auth/google ───────────────────────────────────────────────────────
router.post(
  '/google',
  googleLimiter,
  validate(googleAuthSchema),
  async (req: Request, res: Response, next: NextFunction) => {
    try {
      const { accessToken, refreshToken, user } = await authService.googleAuth(
        req.body.idToken,
        getMeta(req),
      );

      if (!isMobile(req)) {
        res.cookie('refresh_token', refreshToken, REFRESH_COOKIE_OPTIONS);
        return res.json({ accessToken, user });
      }

      return res.json({ accessToken, refreshToken, user });
    } catch (err) {
      next(err);
    }
  },
);

// ─── POST /auth/logout ───────────────────────────────────────────────────────
// Revokes the current session's refresh token and clears the cookie.
// No refresh token needed — just the access token to identify who's logging out.
router.post(
  '/logout',
  authenticate,
  async (req: AuthRequest, res: Response, next: NextFunction) => {
    try {
      // tokenId is embedded in the JWT payload by our token generation
      const tokenId = req.user?.tokenId;
      await tokensService.revokeUserTokens(req.user!.sub, tokenId);

      res.clearCookie('refresh_token', { path: '/api/v1/auth' });
      return res.json({ message: 'Logged out successfully' });
    } catch (err) {
      next(err);
    }
  },
);

// ─── GET /auth/me ────────────────────────────────────────────────────────────
// Returns the authenticated user's profile. Used to restore session on page load.
router.get(
  '/me',
  authenticate,
  async (req: AuthRequest, res: Response, next: NextFunction) => {
    try {
      const user = await usersService.findById(req.user!.sub);
      if (!user) return res.status(404).json({ message: 'User not found' });

      const { passwordHash, ...safe } = user as any;
      return res.json({ user: safe });
    } catch (err) {
      next(err);
    }
  },
);

export { router as authRouter };
```

## 3. Response Shapes

### POST /auth/login — Web
```json
200 OK

// Body
{ "accessToken": "eyJhbGci...", "user": { ... } }

// Cookie (automatic)
Set-Cookie: refresh_token=...; HttpOnly; Secure; SameSite=Strict; Path=/api/v1/auth
```

### POST /auth/login — Mobile
```json
200 OK
{
  "accessToken": "eyJhbGci...",
  "refreshToken": "eyJhbGci...",
  "user": {
    "id": "64a1b...",
    "email": "user@example.com",
    "firstName": "John",
    "provider": "local",
    "emailVerified": false
  }
}
```

### POST /auth/google
```json
200 OK
// Same shape as login — web gets cookie, mobile gets refreshToken in body
{
  "accessToken": "eyJhbGci...",
  "user": {
    "id": "64a1b...",
    "email": "user@gmail.com",
    "firstName": "John",
    "avatarUrl": "https://lh3.googleusercontent.com/...",
    "provider": "google",
    "emailVerified": true
  }
}
```

### Errors
```json
// Wrong password or email
401 Unauthorized
{ "message": "Invalid credentials" }

// Google user trying to login with credentials
400 Bad Request
{ "message": "This account was created with Google. Please sign in with Google." }

// Local user trying to login with Google (same email)
409 Conflict
{ "message": "An account with this email already exists. Please log in with your password." }

// Rate limited
429 Too Many Requests
{ "message": "Too many login attempts, try again later" }
```

## 4. Frontend — Google Button Integration

### Next.js (Web)
```tsx
// Using @react-oauth/google
import { GoogleLogin } from '@react-oauth/google';

export function GoogleLoginButton() {
  const { loginWithGoogle } = useAuth();

  return (
    <GoogleLogin
      onSuccess={async (credentialResponse) => {
        // credentialResponse.credential IS the id_token — send it directly to backend
        if (credentialResponse.credential) {
          await loginWithGoogle(credentialResponse.credential);
        }
      }}
      onError={() => {
        console.error('Google login failed');
      }}
    />
  );
}
```

### React Native (Mobile)
```typescript
// Using @react-native-google-signin/google-signin
import { GoogleSignin } from '@react-native-google-signin/google-signin';

GoogleSignin.configure({ webClientId: 'YOUR_GOOGLE_CLIENT_ID' });

async function handleGoogleLogin() {
  await GoogleSignin.hasPlayServices();
  const userInfo = await GoogleSignin.signIn();

  // userInfo.data.idToken IS the id_token
  const idToken = userInfo.data?.idToken;
  if (!idToken) throw new Error('No id_token received from Google');

  // Send to your backend
  await loginWithGoogle(idToken);
}
```

## 5. Key Security Notes

**Why single "Invalid credentials" error?**
Never tell the user whether the email or password was wrong. If you say "email not found", attackers use it to enumerate registered emails. Generic message = no information leakage.

**Why run bcrypt even when user is not found?**
`bcrypt.compare` takes ~300ms. If you skip it for non-existent users, your endpoint returns instantly for fake emails and slowly for real ones. Timing difference = email enumeration.

**Why reject Google→local account linking silently?**
Auto-linking accounts without explicit user confirmation is a known OAuth account takeover vector. Force the user to consciously connect their accounts through a dedicated "link accounts" flow after authenticating.
