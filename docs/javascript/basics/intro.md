# Introduction to JavaScript

JavaScript is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions. It is most well-known as the scripting language for Web pages, but it is also used in many non-browser environments such as Node.js.

## What You'll Learn

In this section, we'll cover the absolute fundamentals that every JS developer needs to know.

- Variables and data types
- Control flow and loops
- Functions and scope
- The event loop

## A Simple Example

```javascript
const greet = (name) => {
  return `Hello, ${name}!`;
};

console.log(greet("World"));
```

JavaScript runs in every browser, making it the universal language of the web. Whether you're building interactive UIs, server-side APIs, or mobile apps — JavaScript is at the core.
